# octopus-ui
| Step                             | Command / Action                                                                                                                                         | Description                                                                       |
| -------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------- | --------------------------------------------------------------------------------- |
| 1️⃣ Install Node.js & npm        | `node -v` / `npm -v`                                                                                                                                     | Check installation. If missing, install LTS from [nodejs.org](https://nodejs.org) |
| 2️⃣ Create React project         | `npx create-react-app octopus-ui`<br>`cd octopus-ui`                                                                                                     | Initializes React app folder                                                      |
| 3️⃣ Replace boilerplate code     | Copy generated `App.jsx`, `index.jsx`, `components/`, `styles/` into `src/`                                                                              | Replace default CRA files                                                         |
| 4️⃣ Install dependencies         | `npm install react-flow-renderer react-dnd react-dnd-html5-backend`                                                                                      | Required for drag-and-drop canvas                                                 |
| 5️⃣ Run locally                  | `npm start`                                                                                                                                              | Opens `http://localhost:3000` in browser, supports hot reload                     |
| 6️⃣ Build for production         | `npm run build`                                                                                                                                          | Generates `build/` folder with static files                                       |
| 7️⃣ Deploy online - Vercel       | `npm install -g vercel`<br>`vercel`                                                                                                                      | Deploys app online quickly                                                        |
| 7️⃣ Deploy online - Netlify      | Drag-and-drop `build/` folder on Netlify dashboard                                                                                                       | Alternative online deployment                                                     |
| 7️⃣ Deploy online - GitHub Pages | `npm install gh-pages --save-dev`<br>Add scripts in `package.json`:<br>`"predeploy": "npm run build", "deploy": "gh-pages -d build"`<br>`npm run deploy` | Publishes site to GitHub Pages                                                    |
